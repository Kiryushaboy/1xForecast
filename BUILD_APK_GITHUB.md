# Сборка APK через GitHub Actions (БЕЗ прав администратора)

## Проблема
- Нет прав администратора для включения Developer Mode
- Flutter требует symlinks для сборки с плагинами
- Локальная сборка невозможна

## Решение: GitHub Actions

### Шаг 1: Запушьте workflow
Файл уже создан: `.github/workflows/build-apk.yml`

Запушьте его:
```bash
git add .github/workflows/build-apk.yml
git commit -m "Добавлен GitHub Actions для сборки APK"
git push
```

### Шаг 2: Запустите сборку на GitHub

1. Перейдите на GitHub: https://github.com/Kiryushaboy/1xForecast
2. Откройте вкладку **Actions**
3. Выберите **Build APK** в списке слева
4. Нажмите **Run workflow** (справа вверху)
5. Нажмите зелёную кнопку **Run workflow**

### Шаг 3: Дождитесь сборки

- Сборка займёт ~5-10 минут
- Статус будет виден в реальном времени
- Вы увидите зелёную галочку ✅ когда всё готово

### Шаг 4: Скачайте APK

1. Откройте завершённый workflow (кликните на него)
2. Внизу страницы будет секция **Artifacts**
3. Скачайте **release-apk.zip**
4. Распакуйте - там будет 3 APK файла:
   - `app-arm64-v8a-release.apk` ← Для вашего телефона
   - `app-armeabi-v7a-release.apk` (старые устройства)
   - `app-x86_64-release.apk` (эмуляторы)

### Шаг 5: Установите на телефон

1. Скопируйте `app-arm64-v8a-release.apk` на телефон
2. Откройте файл на телефоне
3. Разрешите установку из неизвестных источников (если попросит)
4. Установите

## Автоматическая сборка

Workflow настроен так, что APK будет собираться автоматически при каждом push в main ветку.

## Альтернатива: Codemagic

Если GitHub Actions не подходит, можно использовать Codemagic:
https://codemagic.io/start/
(Бесплатно 500 минут сборки в месяц)
